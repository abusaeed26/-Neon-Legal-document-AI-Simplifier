# ⚡ Neon Legal Document AI Simplifier

An AI-powered web app that simplifies complex legal documents into easy-to-understand language using LLMs.

---

## 🚀 Features

- 📄 Upload **PDF, DOCX, or Image**
- 🧠 Extract text automatically (OCR supported)
- ⚖️ Simplify legal language into plain English
- 🔍 Detect key clauses (Payment, Termination, Liability)
- ⚠️ Highlight risks and important points
- 🌈 Beautiful **Neon UI (Gradio)**

---

## 🛠️ Tech Stack

- Python 🐍
- Gradio (UI)
- OpenRouter API (LLM)
- PyPDF2 (PDF parsing)
- python-docx (DOCX reading)
- pytesseract (OCR for images)

---

## 🧠 Model Used

- `openai/gpt-oss-120b:free` (via OpenRouter)
